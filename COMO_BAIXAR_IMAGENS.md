# Como Baixar Imagens do Site Original

## Método 1: Extract.pics (Mais Fácil) ⭐

1. Acesse: https://extract.pics/
2. Cole a URL: `https://bastardsbrewery.com.br/`
3. Clique em "Extract"
4. A ferramenta mostrará todas as imagens do site
5. Selecione as imagens que deseja e clique em "Download"
6. Organize as imagens nas pastas criadas:
   - `public/images/cervejas/` - Imagens dos rótulos
   - `public/images/fundadores/` - Fotos dos fundadores
   - `public/images/pubs/` - Imagens dos pubs
   - `public/images/` - Logo e outras imagens gerais

## Método 2: DevTools do Navegador

1. Abra o site: https://bastardsbrewery.com.br/
2. Pressione `F12` para abrir o DevTools
3. Vá na aba "Network" (Rede)
4. Filtre por "Img" (imagens)
5. Recarregue a página (`F5`)
6. Você verá todas as imagens carregadas
7. Clique com botão direito em cada imagem e "Save as..."
8. Salve nas pastas correspondentes

## Método 3: Inspecionar Elemento

1. Abra o site no navegador
2. Clique com botão direito na imagem desejada
3. Selecione "Inspecionar elemento" ou "Inspect"
4. Encontre a tag `<img>` ou o CSS `background-image`
5. Copie a URL da imagem
6. Cole no navegador e baixe a imagem
7. Salve na pasta correta

## Estrutura de Pastas Criada

```
public/images/
├── cervejas/          ← Rótulos das cervejas
├── fundadores/        ← Fotos dos fundadores (Xico, Humberto, Richard)
├── pubs/              ← Imagens dos pubs
├── outros/            ← Outras imagens
└── logo.png           ← Logo da empresa (na raiz)
```
